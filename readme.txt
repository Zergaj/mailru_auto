Для запуска теста необходимо установить pytest и selenium.

Команда запуска из директории /tests

pytest test_send_email.py

Архитектура:

- Каждая страница mail.ru вынесена в отдельный класс. Название совпадает с url страницы.
- Composer iframe тоже вынесен в отдельный класс editor_iframe.
- Тесты параметризированы. Для этого созданы отдельные модели email и user. Чтобы изменить тестовые данные,
отредактируйте значения в этих классах или добавьте новые classmethods и затем передайте их в тесты.
